# 🔧 トラブルシューティング：プロジェクトが見つからない

## 問題：Vercelでプロジェクトが見つからない

### 確認事項1: GitHubリポジトリが公開されているか

1. **GitHubでリポジトリを確認**
   - https://github.com にアクセス
   - あなたのリポジトリを開く
   - リポジトリ名の横に「Public」と表示されているか確認
   - 「Private」になっている場合は、設定を変更する必要があります

2. **リポジトリを公開する方法**
   - リポジトリページで「Settings」をクリック
   - 下にスクロールして「Danger Zone」セクションを見つける
   - 「Change visibility」→「Change to public」をクリック
   - リポジトリ名を入力して確認

### 確認事項2: VercelでGitHubを接続しているか

1. **Vercelのダッシュボードを確認**
   - https://vercel.com/dashboard にアクセス
   - 左側のメニューで「Settings」→「Git」をクリック
   - GitHubが接続されているか確認

2. **GitHubを接続する方法**
   - 「Connect Git Provider」をクリック
   - 「GitHub」を選択
   - 認証を完了

### 確認事項3: リポジトリが表示されない場合

1. **手動でリポジトリを追加**
   - Vercelのダッシュボードで「Add New...」→「Project」をクリック
   - 「Import Git Repository」セクションで
   - 「Configure GitHub App」または「Grant Access」をクリック
   - リポジトリへのアクセス権限を付与

2. **リポジトリを検索**
   - 「Search repositories」でリポジトリ名を検索
   - 見つからない場合は、上記の手順で権限を付与

### 確認事項4: リポジトリ名を確認

1. **GitHub Desktopで確認**
   - GitHub Desktopを開く
   - 「Repository」→「View on GitHub」をクリック
   - ブラウザでリポジトリが開く
   - URLからリポジトリ名を確認（例: `https://github.com/username/soccer-standings`）

2. **Vercelで正確な名前で検索**
   - リポジトリ名を正確に入力して検索

## 🚀 解決方法：手動でインポート

### ステップ1: Vercelで新規プロジェクトを作成

1. **Vercelのダッシュボードにアクセス**
   - https://vercel.com/dashboard

2. **「Add New...」→「Project」をクリック**

3. **「Import Git Repository」を確認**
   - GitHubアカウントが表示されているか確認
   - 表示されない場合は「Configure GitHub App」をクリック

### ステップ2: GitHubアプリを設定

1. **「Configure GitHub App」をクリック**
   - GitHubの認証ページが開く

2. **リポジトリへのアクセスを許可**
   - 「Only select repositories」を選択
   - あなたのリポジトリを選択
   - または「All repositories」を選択（簡単）

3. **「Install」をクリック**

### ステップ3: リポジトリをインポート

1. **Vercelに戻る**
   - ページをリロード（F5）

2. **リポジトリを検索**
   - 「Search repositories」でリポジトリ名を入力
   - または一覧から選択

3. **「Import」をクリック**

## 📋 チェックリスト

問題を解決するために、以下を確認してください：

- [ ] GitHubリポジトリが「Public」になっている
- [ ] VercelでGitHubアカウントが接続されている
- [ ] GitHubアプリでリポジトリへのアクセス権限が付与されている
- [ ] リポジトリ名を正確に入力して検索している

## 💡 それでも見つからない場合

### 方法1: GitHubのURLから直接インポート

1. **GitHubでリポジトリを開く**
   - リポジトリのURLをコピー（例: `https://github.com/username/soccer-standings`）

2. **Vercelで「Import Git Repository」**
   - 「Or paste a repository URL」をクリック
   - URLを貼り付けて「Import」をクリック

### 方法2: Vercel CLIを使う（上級者向け）

ターミナルで以下を実行：

```bash
# Vercel CLIをインストール
npm i -g vercel

# プロジェクトディレクトリに移動
cd /Users/kazuyaishibashi/Development/20251115

# デプロイ
vercel
```

## 🆘 まだ解決しない場合

1. **Vercelのサポートに問い合わせ**
   - https://vercel.com/support

2. **GitHubのリポジトリ設定を確認**
   - Settings → Collaborators で権限を確認

3. **ブラウザのキャッシュをクリア**
   - Cmd + Shift + R で強制リロード
