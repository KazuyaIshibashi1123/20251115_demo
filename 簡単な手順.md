# 🚀 初心者でもできる！簡単な公開手順

## ステップ1: GitHub Desktopをインストール（5分）

### 1. ダウンロード
1. ブラウザで https://desktop.github.com/ を開く
2. 「Download for macOS」をクリック
3. ダウンロードしたファイルを開いてインストール

### 2. ログイン
1. GitHub Desktopを起動
2. 「Sign in to GitHub.com」をクリック
3. ブラウザが開くので、GitHubアカウントでログイン
   - アカウントがない場合は「Sign up」で新規登録（無料）

## ステップ2: GitHubにプッシュ（3分）

### 1. リポジトリを追加
1. GitHub Desktopで「File」→「Add Local Repository」をクリック
2. 「Choose...」をクリック
3. `/Users/kazuyaishibashi/Development/20251115` フォルダを選択
4. 「Add repository」をクリック

### 2. コミット
1. 左側に「Uncommitted changes」と表示される
2. すべてのファイルにチェックが入っていることを確認
3. 下部の「Summary」に「サッカー順位表アプリ」と入力
4. 「Commit to main」をクリック

### 3. 公開
1. 上部の「Publish repository」をクリック
2. 「Name」に「soccer-standings」と入力
3. 「Keep this code private」のチェックを**外す**（公開するため）
4. 「Publish repository」をクリック

✅ **完了！** GitHubに公開されました！

## ステップ3: Vercelでデプロイ（5分）

### 1. Vercelに登録
1. ブラウザで https://vercel.com を開く
2. 「Sign Up」をクリック
3. 「Continue with GitHub」をクリック
4. GitHubアカウントでログイン
5. 「Authorize Vercel」をクリック

### 2. プロジェクトをインポート
1. ダッシュボードで「Add New...」→「Project」をクリック
2. 「Import Git Repository」で「soccer-standings」を選択
3. 「Import」をクリック

### 3. 設定（そのままでOK）
- **Framework Preset**: Other（そのまま）
- **Root Directory**: `./`（そのまま）
- **Build Command**: （空欄のまま）
- **Output Directory**: （空欄のまま）
- **Install Command**: `npm install`（そのまま）

### 4. デプロイ
1. 「Deploy」をクリック
2. 2〜3分待つ
3. 「Congratulations!」と表示されたら完了！

✅ **完了！** あなたのアプリが公開されました！

### 5. URLを確認
- 「Visit」ボタンをクリックすると、あなたのアプリが開きます
- URLは `https://soccer-standings-xxxxx.vercel.app` のような形式です
- このURLを誰かに共有できます！

## 🎉 完成！

これで、世界中の誰でもあなたのアプリにアクセスできます！

### 今後の更新方法

## ❓ よくある質問

### Q: 無料で使えますか？
A: はい！GitHubもVercelも無料プランで十分使えます。

### Q: サーバーは動き続けますか？
A: はい。Vercelの無料プランでも24時間動き続けます。

### Q: データは最新ですか？
A: はい。ページを開くたびにYahooスポーツから最新データを取得します。

### Q: エラーが出たら？
A: Vercelのダッシュボードで「Deployments」を確認し、ログを見てください。
